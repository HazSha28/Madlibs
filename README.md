# Madlibs Story Generator (Python + Tkinter)

A fun and interactive MadLib generator built using Python with a clean Tkinter UI.  
You can pick templates, fill placeholders, random-fill, preview, copy, and save stories.

---
##  Features
- Multiple story templates  
- Auto-detected placeholders  
- Random fill button  
- Live story preview  
- Save generated story as `.txt`  
- Clean UI with Tkinter  
- Beginner-friendly & GitHub-ready structure
  
---
##  Requirements
Python 3.8+  
Tkinter (built into Python)
Install additional dependencies
